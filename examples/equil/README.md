# equil

The first runnable target uses the existing MATSim `equil` scenario.

Recommended config source:

- `/home/kentaro/repos/matsim-libs/examples/scenarios/equil/config.xml`
- `/home/kentaro/repos/matsim-example-project/scenarios/equil/config-benchmark.xml`

Example:

```bash
cargo run -p matsim-cli -- run --config /home/kentaro/repos/matsim-example-project/scenarios/equil/config-benchmark.xml
```
